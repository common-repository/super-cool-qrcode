Easily insert QR codes in your blog, with Widget or Shortcode.

== Description ==

The *Super Cool QRCode* use the [Google Chart API](http://code.google.com/apis/chart/ "Google Chart API") to generate on-the-fly [QR codes](http://en.wikipedia.org/wiki/QR_Code "QR Codes") for your blog. You can encode any type of strings that can be readed in a large array of capable devices.

To generate QR codes you can use:

* A Widget
* A Shortcode

See the new [Super Cool QRCode homepage](http://www.incerteza.org/blog/projetos/super-cool-qrcode/ "Super Cool QRCode") for further information.

Thanks for everyones suggestions!

== Installation ==

1. Upload the full directory into your /wp-content/plugins/ directory, or install it through the admin interface
2. Use the widget or the shortcode
3. DONE!
